# Chatbot UASD - Estatuto Orgánico

## Instalación

```bash
npm install
```

Crea un archivo `.env` basado en `.env.example`:

```env
OPENAI_API_KEY=tu_api_key
PORT=3000
MODEL=gpt-5.5
```

Descarga el PDF oficial del Estatuto Orgánico de la UASD y colócalo aquí:

```txt
server/data/estatuto.pdf
```

## Ejecutar

```bash
npm run dev
```

Abre:

```txt
http://localhost:3000
```
